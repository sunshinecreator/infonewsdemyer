# 🧼 蚊子修改器

这是一个简单的网页工具，用于直白的表述。

## 💡 功能亮点

- 输入任意文本，自动识别并替换“新词”
- 支持自定义替换词库
- 纯 HTML/CSS/JS 实现，无需服务器

## 🔧 技术栈

- HTML5
- CSS3
- JavaScript (Fetch API)
- JSON 数据驱动

## 🚀 使用方法

1. 克隆本项目
2. 打开 `index.html` 即可使用
3. 修改 `data/mapping.json` 添加你自己的替换规则
